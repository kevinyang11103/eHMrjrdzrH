# 校园订餐小程序+SSM

## 前言

随着互联网的快速发展，校园生活越来越便捷。本团队基于SSM框架和微信小程序开发了一款校园订餐应用，旨在为广大师生提供方便、快捷的订餐体验。以下是项目详细介绍。

## 内容介绍

本项目是一款集餐厅菜品展示、用户下单、订单管理等功能于一体的校园订餐小程序。用户可以通过微信小程序浏览餐厅菜品，完成下单、支付等操作。后台管理端则负责处理订单、维护菜品信息等。通过本项目，我们希望为校园生活带来更多便利。

## 技术介绍

- 语言：Java
- 使用框架：Spring、Spring MVC、MyBatis、微信小程序
- 前端技术：JS、Vue、CSS3、Uniapp
- 开发工具：IDEA/Eclipse、Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12、14、16
